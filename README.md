# time-tales
Time-Tales is a small browser game for young kids to practice telling time with an analog clock.

The current version is a single-file HTML mini app for first-grade learners:
- It generates 5 random tasks per round.
- It mixes two task types: reading an analog clock and setting an analog clock.
- Times are limited to 5-minute steps.
- Reading tasks can include morning or afternoon, so kids answer with the matching 24-hour time when needed.
- If the player gets at least 4 out of 5 correct, the app shows a random reward image from the `images` folder.

Everything runs from `index.html`, so the game can be opened directly in a browser on desktop or mobile.
